驱动器的细分设置由拨码开关的SW1~SW3来设定，默认为2细分，一般的两相四线制步进电机的步进角都是1.8°， 因此电机旋转一圈需要360° /1.8° =200个脉冲，这里2细分转一圈 需要的脉冲数为200*2=400个。具体详见下表：

| 细分 | 脉冲  | SW1  | SW2  | SW3  |
| ---- | ----- | ---- | ---- | ---- |
| 2    | 400   | OFF  | OFF  | OFF  |
| 4    | 800   | ON   | OFF  | OFF  |
| 8    | 1600  | OFF  | ON   | OFF  |
| 16   | 3200  | ON   | ON   | OFF  |
| 32   | 6400  | OFF  | OFF  | ON   |
| 64   | 12800 | ON   | OFF  | ON   |
| 128  | 25600 | OFF  | ON   | ON   |
| 256  | 51200 | ON   | ON   | ON   |
