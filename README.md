# bookReparation

## 介绍
该系统利用摄像头模块对工作台上平放的残缺古籍进行边缘检测，计算得出物理坐标位置。上位机软件控制机械传动机构运动，执行补胶贴纸等动作。

**系统设计与实现**
下位机：主控stm32、系统：freertos
上位机：imx6ull、系统：Linux。

## Bsp工程包
包括电机控制板级支持包等。

## 端口定义
RELAY0: 吸盘(吸取补纸) 
RELAY1: 喷笔(喷水打湿补纸) 
RELAY2: 摄像头补光灯
RELAY3: 激光12V开关。

PINB1: 激光pwm输出引脚。

串口1：下位机与pc通信端口。 串口2：下位机与ardunio通信接口。
